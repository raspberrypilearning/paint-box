## Изазов: Пречице

Can you create keyboard shortcuts for your commands? На пример:

+ б = промени у плаву бојицу
+ г = промени у зелену бојицу
+ е = промени гумицу
+ ц = обриши екран

You could even allow the user to change the pen width with the arrow keys.
