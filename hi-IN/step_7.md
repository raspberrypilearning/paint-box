## चुनौती: शॉर्टकट्स

क्या आप अपने कमांड्स(commands) के लिए कीबोर्ड शॉर्टकट्स बना सकते हैं? उदाहरण के लिए:

+ b = Switch to blue pen
+ g = switch to green pen
+ e = switch to eraser
+ c = clear screen

You could even allow the user to change the pen width with the arrow keys.
